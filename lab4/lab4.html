getDate(): Devuelve el día del mes (1-31) para la fecha especificada.
getMonth(): Devuelve el mes (0-11) para la fecha especificada.
getFullYear(): Devuelve el año (cuatro dígitos) para la fecha especificada.
getHours(): Devuelve la hora (0-23) para la fecha especificada.
toISOString(): Devuelve una cadena en formato ISO (YYYY-MM-DDTHH:mm:ss.sssZ) que representa la fecha.

push(): Agrega uno o más elementos al final de un arreglo y devuelve la nueva longitud del arreglo.
pop(): Elimina el último elemento de un arreglo y lo devuelve.
splice(): Cambia el contenido de un arreglo eliminando o reemplazando elementos existentes y/o agregando nuevos elementos.
indexOf(): Devuelve el primer índice en el que se encuentra un elemento especificado en un arreglo.
forEach(): Ejecuta una función para cada elemento en un arreglo.

Para declarar una variable con alcance local dentro de una función en JavaScript, utiliza la palabra clave let o const seguida del nombre de la variable y opcionalmente asigna un valor inicial.

Falta de encapsulamiento: Las variables globales son accesibles desde cualquier parte del código, lo que puede dificultar el mantenimiento y la comprensión del código.
Posible contaminación: Si una función modifica una variable global, puede afectar el estado de otras partes del programa, lo que puede generar efectos secundarios no deseados.
Dependencia externa: Las funciones que utilizan variables globales dependen de un estado externo que puede cambiar en cualquier momento, lo que puede hacer que el código sea menos predecible y propenso a errores.
